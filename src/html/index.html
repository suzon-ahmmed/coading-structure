A) Installation

When you only need to include VueStudio compiled CSS or JS, below is the required core css and javascript.

                        
<-- core css -->
<link href="assets/css/vendor.min.css" rel="stylesheet" />
<link href="assets/css/app.min.css" rel="stylesheet" />
 
<-- core js -->
<script src="assets/js/vendor.min.js"></script>
<script src="assets/js/app.min.js"></script>
B) File Structure

File structure overview for VueStudio.

                        
template/
├── dist/             // generated html / css / js files
├── gulpfile.js       // gulp settings
├── package.json      // npm package installed
└── src/
    ├── html/         // html source files
    ├── img/          // image source files
    ├── js/           // javascript source files
    └── scss/         // VueStudio scss files
C) HTML Structure

Below is the general page structure and the required core css / js files for VueStudio Admin Template. The css / js files can be found via /dist/assets/ folder.

              
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8" />
  <title>VueStudio</title>
  <meta content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" 
        name="viewport" />
 
  <!-- core-css -->
  <link href="assets/css/vendor.min.css" rel="stylesheet" />
  <link href="assets/css/app.min.css" rel="stylesheet" />
</head>
<body>
  <div id="app" class="app">
    <!-- app-header -->
    <div id="header" class="app-header">... </div>
    
    <!-- app-sidebar -->
    <div id="sidebar" class="app-sidebar">... </div>
    
    <!-- app-content -->
    <div id="content" class="app-content">...</div>
    
    <!-- btn-scroll-top -->
    <a href="#" data-click="scroll-top" class="btn-scroll-top fade"><i class="fa fa-arrow-up"></i></a>
  </div>
  
  <!-- BEGIN core-js -->
  <script src="assets/js/vendor.min.js"></script>
  <script src="assets/js/app.min.js"></script>
</body>
</html>
